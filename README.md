# AS Topology Visualizer (Chile)

Visualización interactiva de la topología de Internet a nivel de Sistemas Autónomos (AS)
construida a partir de datos BGP y RIPE Atlas.

## Features
- Métricas globales AS-level
- Top-K ASNs más conectados
- Exploración local (ego-network por ASN)
- Subgrafos centrados en ASNs
- Visualización vía Streamlit

## Run
```bash
streamlit run app/app.py
